# The Spark Engineering Story

A blog documenting the journey of building **Spark** — a software engineering
organisation that happens to have one human engineer — and the principles discovered
along the way.

**This repository is Product 1 of the Spark factory.** Its first article was planned,
generated, quality-checked, built, and published by running the whole chain *through*
Spark (local Ollama inference for generation; Spark's Execution Engine / `gh` for
repository creation). Product code lives here, never inside Spark (Constitution 15).

- `content/` — brief + Markdown posts (`posts/` generated by Spark inference)
- `tools/` — the production run: `generate.py` · `quality_check.py` · `build.py` · `publish.py`
- `docs/` — built static site (served by GitHub Pages: `main:/docs`)
- `DELIVERY_001.md` — the delivery record + two-question scorecard

Published site: https://rhyscraig.github.io/spark-engineering-blog/
